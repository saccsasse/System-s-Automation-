Systém ALMAS je heterogenní MAS, který simuluje provoz futuristické přepravní společnosti. Skládá se ze tří typů agentů: Heavy-Truck (dálkový tahač), City-Van (městský doručovatel) a Hub-Manager (statický agent skladu). Agenti Hub-Manager vypisují zakázky (přeprava zboží). Mobilní agenti (Truck/Van) o tyto zakázky soutěží v aukcích. Unikátním prvkem systému je schopnost dálkových tahačů formovat platooning (konvoje) pro úsporu energie. Agenti se musí dynamicky domlouvat, kdo konvoj povede (vyšší spotřeba) a kdo se poveze v závěsu (úspora), přičemž si sdílejí ušetřené náklady. Systém zahrnuje i nutnost pravidelné údržby a dobíjení, což vnáší prvek rizika – pokud agent špatně naplánuje energii, uvízne a platí vysokou pokutu za odtah
